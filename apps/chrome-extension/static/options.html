Pulsarbot Chrome Extension Executor

Use a dedicated Chrome profile when possible. Pairing is short-lived, and attach is always explicit.

Recommended Flow

  1. Build the repo and load this extension unpacked from apps/chrome-extension/dist.
  2. Create a Chrome Extension executor in Pulsarbot.
  3. Pair in the popup, then click Attach Current Window while the target browser window is focused.
  4. Keep that attached window on the site whose login state you want the agent to use.
