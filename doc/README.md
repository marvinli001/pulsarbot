# Pulsarbot 文档总览

这个目录用于承载 Pulsarbot 当前仓库的使用文档、开发文档和排障文档。它的目标不是复述计划，而是基于当前代码库的实际落地状态，给出一套可以直接使用的说明。

## 建议阅读顺序

1. [快速开始](quick-start.md)：本地安装、构建、启动与首次联调入口
2. [管理台使用说明](miniapp-guide.md)：Telegram Mini App 中各个面板的职责和推荐操作顺序
3. [仓库结构与架构](project-structure.md)：理解 monorepo、模块边界和运行时数据流
4. [API 总览](api-overview.md)：查看服务端关键接口
5. [开发、测试与运维](development-and-ops.md)：开发命令、测试入口、部署与日常维护
6. [常见问题与排障](help-and-faq.md)：处理本地联调和部署阶段常见问题

## 文档定位

- 面向使用者：关注如何完成 bootstrap、配置 provider、启用能力、排查运行状态
- 面向开发者：关注目录结构、模块职责、命令、测试、部署与接口边界
- 面向运维者：关注 `DATA_DIR`、健康检查、日志、作业重试、导入导出恢复

## 当前原则

- 只写仓库里已经存在的能力
- 能给出命令时优先给出命令
- 能给出路径时优先给出路径
- 未来规划请参考根目录计划文档，不混写到这里
